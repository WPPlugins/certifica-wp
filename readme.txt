Este plugin permite incluir botões para gerar certificados para os alunos do seu curso online.

== Description ==

Além de gerar os certificados, é possível utilizar o validador para confirmar a autenticidade de um certificado.

Com apenas alguns cliques você pode gerar certificados para os alunos dos seus cursos online. Tudo de forma automática e simples.

Certificados em PDF, alta qualidade e exclusivos para o seu curso!

== Installation ==

1. Instale o plugin normalmente no seu Wordpress.
2. Acesse [http://certificawp.com.br](http://certificawp.com.br) e cadastre-se gratuitamente.
3. Crie o curso e informe os detalhes solicitados.

Para utilizar a integração via plugin Wordpress:

4. Acesse o menu Configurações > Certifica WP para definir o nome que será utilizado na geração do certificado.
5. No seu Wordpress, utilize o shortcode [certificawp_obter key=""] na página onde o botão de download do certificado deverá ser exibido.

Você também pode gerar certificados através da integração customizada via script, por email ou por uma página pública do seu curso.
Acesse [http://certificawp.com.br](http://certificawp.com.br) para ver todos os recursos.
Experimente grátis!
